# GradleTest 🧪

Gradle 构建系统学习与测试项目。

## 🎯 项目简介

本项目用于学习和测试 Gradle 构建系统的各种功能，包括依赖管理、多模块构建等。

## 📂 项目结构

```
GradleTest/
├── app/                 # 应用模块
├── gradle/              # Gradle wrapper
├── build.gradle         # 构建配置
├── settings.gradle      # 项目设置
├── gradle.properties    # Gradle 属性
├── gradlew              # Gradle wrapper (Unix)
└── gradlew.bat          # Gradle wrapper (Windows)
```

## 🔧 主要内容

### 构建配置
- 依赖管理
- 多渠道打包
- 构建变体

### 插件使用
- Java 插件
- Android 插件
- 自定义插件

## 🚀 快速开始

```bash
# 克隆仓库
git clone https://github.com/MengqiYe/GradleTest.git

# 构建项目
./gradlew build

# 运行任务
./gradlew tasks

# 清理构建
./gradlew clean
```
